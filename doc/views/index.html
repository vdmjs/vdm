Vdm

A lightweight directive template solution

Get Started View on GitHub Get Cdn Url

vdm是一个轻量级的指令型模板解决方案

如果你用过angular或者vue，应该对指令（directive）这个概念并不陌生,angular大而全，vue精简后增加了组件机制，而vdm结合angular，react，vue的特点实现了一套专注于解决模板问题的解决方案。

Vdm具有以下特点

  • 单一职责
    vdm只负责解决模板问题。使用者可以单独使用，也可以跟任何其他框架结合使用
  • 更好的语法
    支持类mustache风格的模板语法，很好的避免了指令写多了模板可读性差的问题
  • virtual dom
    具有virtual dom中间层，一方面加快了分析指令的速度，另一方面也为服务端渲染提供了可能
  • 监听机制多样
    同时支持defineProperties与脏检测两种数据检测机制。并且都支持ie8。用户自行选择使用

  • 测试

    采用karma + jasmine做功能性测试

    cd vdm
npm test

  • 打包

    使用webpack打包，在根目录执行

    cd vdm
npm run-script build
